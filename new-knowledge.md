**css**

Научился на практике работать с псевдоклассами и псевдоэлементами.
```bash
/* каждый второй прямой потомок с фоном */
    .news >  :nth-child(even) {
    background-color: #636b6f6c;

}

/* первые два прямых потомка с надписью New */
.news > *:nth-child(-n+2)::before {
    margin-top: 10px;
    content: "New*";
    color: rgb(108, 117, 125);
    font-family: 'Times New Roman', Times, serif;
}
```
---
1. Также узнал про оператор  **">"**
>Оператор > в CSS — это дочерний комбинатор (child combinator). Он выбирает только те элементы, которые являются непосредственными (прямыми) потомками указанного родителя.

В то время, когда (пробел) для любых потомков на любых уровнях вложенности

2. Оператор "*" - Любой тип элемента (div, article, p и т.д.)
> надежнее заменить на .news > article:nth-of-type(-n+2)::before { ... }, так как оператор "*" работает для любого тега, т.е. и для div, article, p ... , что может работать некорректно при изменении html структуры проекта.

3. Самым интересным было именно задание с добавлением New* к первым двум новостям.
>:nth-child(-n+2). Формула -n+2 работает по принципу подстановки значений n, начиная с 0.
n=0 -0+2=2 (Выбираем второй элемент)
n=1 -1+2=1 (Выбираем первый элемент)
n=2 -2+2=0 (Нулевого элемента нет, поэтому дальше ничего не выбираем)
